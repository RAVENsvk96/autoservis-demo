# AutoServis Demo

Moderná webová stránka pre autoservis vytvorená v Next.js.

## Live demo

https://autoservis-demo-beryl.vercel.app

## Funkcie

- responzívny dizajn
- moderný Hero section
- sekcia služieb
- animované štatistiky
- recenzie zákazníkov
- kontaktný formulár
- odosielanie e-mailov cez Resend
- SEO metadata
- Open Graph obrázok
- sitemap.xml
- robots.txt
- deployment cez Vercel

## Technológie

- Next.js
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React
- React CountUp
- Resend
- Vercel
